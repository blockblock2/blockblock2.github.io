One-time setup

Find your Flash Invaders UID

Your UID lets Flashy load which mosaics you've already flashed. You only need to do this once — it's saved in your browser.

Step 1 of 5

Install mitmproxy on your Mac

Open Terminal on your Mac. If you don't have Homebrew, run the first command. Then install mitmproxy.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install mitmproxy
⏱ Homebrew can take a few minutes. Run the second command once it's done.

Tip: If Terminal says "no such file or directory," it's because iPad's auto-correct changed the " quotes into curly quotes. Use the copy button — don't retype.
Step 2 of 5

Start the proxy

Run this in Terminal. It starts listening on port 8080 and opens a web interface.

mitmweb
Terminal — mitmweb
$mitmweb
Web server listening at http://127.0.0.1:8081/
Proxy server listening at http://*:8080

A browser tab may open at 127.0.0.1:8081 — leave it open, you'll need it soon.

Step 3 of 5

Find your Mac's IP address

Run this in a new Terminal tab (mitmweb is still running in the other one).

ipconfig getifaddr en0
Terminal
$ipconfig getifaddr en0
192.168.1.42 ← your Mac's IP, write this down
Step 4 of 5

Set your iPhone's proxy to your Mac

Your iPhone and Mac must be on the same WiFi network.

📶
Settings → Wi-Fi
Tap the ⓘ next to your network name
🔧
Scroll to HTTP Proxy → Configure Proxy
Choose Manual
📝
Server: 192.168.1.42 (your Mac's IP from step 3)
Port: 8080 — then tap Save
🔐
Open Safari on iPhone and go to mitm.it
Tap Apple, install the certificate, then go to Settings → General → VPN & Device Management and trust it
Step 5 of 5

Open Flash Invaders and grab your UID

Open the Flash Invaders app on your iPhone and tap your profile. Then switch to the 127.0.0.1:8081 tab on your Mac and look for a request to api.space-invaders.com.

mitmweb — intercepted request
GET https://api.space-invaders.com/flashinvaders/users?uid=aBcDeFgHiJkLmNoPqRsT

Copy the string after uid= and paste it below.

💡 After you're done, turn the iPhone proxy off: Settings → Wi-Fi → ⓘ → HTTP Proxy → Off
✅ You're all set! Your flashed mosaics are now shown on the map.
